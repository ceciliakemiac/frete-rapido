# Frete Rapido Desafio API

## Requirements to run API

- [Go](https://golang.org/dl/)
- [Docker](https://docs.docker.com/docker-for-mac/install/)

### File

- .env

## How to Run 

- Create a .env file with the necessary [environment variables], you might just make it a copy of .env.example file;
- Run `make docker-build` to run Dockerfile;
- Run `make start-deps` to Start dependencies;
